# Morning — 晨間工作準備總覽

**by Cindy Lin (cindylinlinlin) | Sales**

一句話啟動早晨：輸入 `早安`，自動整理信件、拉出 HubSpot 追蹤清單、在 Google Calendar 建立兩個電話時段。

---

## Skills

| Skill | 觸發詞 | 功能 |
|-------|--------|------|
| `/morning` | `早安` / `morning` / `好早` | 晨間總覽：信件 + 追蹤清單 + 行事曆 |
| `/mail-check` | `整理信件` / `mail check` | 整理未回覆客戶信件 |
| `/call-prep` | `今天要打給誰` / `call prep` | HubSpot 今日追蹤清單 |
| `/call-block` | `callcall` | 自動建立兩個電話時段 |
| `/callcold` | `冷單` / `callcold` | 找出 7–21 天未聯絡的客戶 |
| `/client-research` | `查XX公司` | 拜訪前自動研究客戶背景 |
