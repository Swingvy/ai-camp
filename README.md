# Swingvy AI Camp

> 5 days to permanently change how every team member works.

A hands-on internal training camp for non-engineers to master **Claude Code** — from first install to building production workflows that automate real work.

## What is this?

Swingvy AI Camp is a 5-day program that teaches every team member — regardless of technical background — to use Claude Code as a daily productivity tool. By the end of the camp, each participant will have built and shipped at least one working automation that solves a real problem from their job.

## Who is it for?

This camp is designed for **non-engineers**: Sales, Customer Success, Marketing, Operations, Finance, HR, and even the PM or QA team. No coding experience required — just bring your laptop and your daily work problems.

## Program at a Glance

| Day | Theme | What you'll do |
|-----|-------|----------------|
| **Day 1** | Install & First Conversations | Set up Claude Desktop App, learn GUI vs CLI, create your personal CLAUDE.md, try web search |
| **Day 2** | Connect Your Tools (MCP) | Connect Slack, Google Drive, Jira, HubSpot, and more — then query across tools |
| **Day 3** | Build Your First Skill | Discover skills on [skills.sh](https://skills.sh/), learn skill anatomy, build your own with the Custom Skill Builder |
| **Day 4** | Chaining, Enhancement & Sharing | Chain skills into workflows, enhance with validation and formatting, share via team repo |
| **Day 5** | Ship Your Project (Demo Day) | Build a complete solution, present to the group, merge into team skills repo |

## How it works

Each day is a **Claude Code Skill** itself. Participants simply run the skill for their day and Claude teaches interactively:

```
/day1-install           # Day 1: Install & first conversations
/day2-connect-tools     # Day 2: Connect your work tools via MCP
/day3-first-skill       # Day 3: Build your first skill
/day4-advanced-workflows # Day 4: Chaining & enhancement
/day5-ship-project      # Day 5: Demo day
```

Each session is **3 hours** with a consistent rhythm:
- **Concept** (40 min) — Learn with live demos
- **Hands-on** (80 min) — Build something real
- **Share & Reflect** (30 min) — Demo to the group

## What participants will learn

- **Day 1**: How to talk to Claude effectively, set up a personal profile (CLAUDE.md), use web search for research, and Plan Mode for complex tasks
- **Day 2**: How to connect daily tools (Slack, Google Workspace, Jira, Confluence, HubSpot, Dropbox) via MCP and run cross-tool queries
- **Day 3**: How to create reusable automations (Skills) — custom commands that Claude follows consistently every time
- **Day 4**: How to chain multiple skills into workflows, add error handling and validation, and share skills with the team
- **Day 5**: How to ship a complete project that solves a real work problem

## Class size

**8-15 participants per cohort** — small enough for hands-on support, large enough for group energy and peer learning.

By the end of the camp, every participant will have:
- Claude Code installed and configured for their role

- 1 complete project shipped and demoed
