# Claude Code Internal Training Camp

This is a 5-day training camp for non-engineers to master Claude Code.

## Skills as Curriculum

Each day is a Claude Code Skill. Participants run `/day1-install`, `/day2-connect-tools`, etc. Claude teaches interactively — explaining concepts, guiding hands-on exercises, and quizzing understanding.

## Structure

- Day 1: Install & First Conversations
- Day 2: Connect Your Tools (MCP)
- Day 3: Build Your First Skill
- Day 4: Advanced Skills & Workflows
- Day 5: Ship Your Project (Demo Day)

## Rules

- Each block follows the 2-turn STOP PROTOCOL (Phase A: explain+execute, Phase B: quiz)
- Match the participant's language (Korean or English)
- Tone: friendly, encouraging, patient — these are non-engineers
- For Claude Code questions, use the claude-code-guide agent
- Celebrate small wins
