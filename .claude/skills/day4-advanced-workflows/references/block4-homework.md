# Block 4: Homework — Prepare for Demo Day

## EXPLAIN

Incredible progress! You now know how to chain skills, enhance them, and share with your team.

### What You Accomplished Today

- Chained two skills into a workflow (Skill A → Skill B)
- Enhanced your skill with validation, formatting, or error handling
- Set up a team skills repository and submitted your skill via PR

### What to Do Tonight — Prepare for Demo Day

Tomorrow is **Demo Day** — you'll build a complete project and present it to the group. Here's how to prepare:

### Task 1: Test Your Workflow End-to-End
Run your chained workflow (Skill A → Skill B) three times:
- Does it produce consistent results?
- Are there any steps that fail or produce unexpected output?
- Time it — how long does the full workflow take?

### Task 2: Add One More Enhancement
Pick one upgrade you haven't tried yet:

| Enhancement | What it does |
|-------------|-------------|
| Input validation | Checks that required data exists before running |
| Output formatting | Makes results look professional (tables, headers, sections) |
| Error messages | Gives helpful feedback when something goes wrong |
| Status updates | Tells the user what's happening at each step |

### Task 3: Write Your Project Pitch
Tomorrow you'll present a 3-minute demo. Start thinking about:

1. **Problem** (30 sec): What task is painful or repetitive in your job?
2. **Solution** (30 sec): What does your skill/workflow do?
3. **Demo** (1.5 min): Show it working live
4. **Impact** (30 sec): How much time does it save? What errors does it prevent?

Write a 2-sentence pitch:
```
"I spend [X hours/week] on [painful task].
My skill automates this by [what it does], saving [time/effort]."
```

### Task 4: Push Your Latest Skill to the Team Repo
Make sure your latest version is committed and pushed:
- Include your enhanced skill and any chained skills
- Update the skill description if you made changes today

### Why This Matters for Day 5

Tomorrow is your chance to:
- Build something complete and polished
- Show the team what you've built
- Contribute to the team's shared skill library

Come with a clear idea and a working foundation — Day 5 is about polishing and presenting, not starting from scratch.

## EXECUTE

Complete the 4 homework tasks above before Day 5.
