# Block 6: Homework — Before Day 4

## EXPLAIN

Amazing work today! You built your first Skill — a reusable automation that saves you real time.

### What You Accomplished

- Discovered existing skills on skills.sh
- Learned the anatomy of a SKILL.md file
- Built your own skill using the Custom Skill Builder
- Tested, iterated, and demoed to the group

### What to Practice Tonight

Before Day 4, complete these tasks to prepare for skill chaining and enhancement:

### Task 1: Run Your Skill 3 More Times
Run your skill with different inputs or scenarios:
- Try it with edge cases — what happens with unusual input?
- Try it with different data — does it still produce good output?
- Note any failures or unexpected behavior

### Task 2: Write Down Edge Cases
Create a list of things that could go wrong:
- What if the data source is empty?
- What if the input is in an unexpected format?
- What if a connected tool is unavailable?

This list will help you add error handling in Day 4.

### Task 3: Think About a Second Skill
Tomorrow you'll chain two skills together. Think about:
- What would be a good **follow-up** to your current skill?
- What task naturally comes **after** what your skill produces?

**Common patterns:**

| Your current skill does... | A second skill could... |
|---------------------------|------------------------|
| Collects data | Formats it into a report |
| Drafts a message | Reviews tone and grammar |
| Gathers feedback | Categorizes and prioritizes |
| Creates a summary | Sends it to the right channel |

### Task 4: Browse skills.sh for Inspiration
Go back to https://skills.sh/ and look for:
- Skills that complement yours
- Enhancement techniques you hadn't considered
- Skills from other roles that inspire new ideas

### Why This Matters for Day 4

Tomorrow you'll learn to:
- **Chain** skills together (Skill A → Skill B)
- **Enhance** skills with validation, formatting, and error handling
- **Share** skills with your team via a git repository

Coming prepared with edge cases and a second skill idea will make Day 4 much more productive.

## EXECUTE

Complete the 4 homework tasks above before Day 4.

**Quick checklist:**
- [ ] Run your skill 3 more times with different inputs
- [ ] Write down 3+ edge cases or potential failures
- [ ] Sketch an idea for a second skill that chains with your first
- [ ] Browse skills.sh for 10 minutes for inspiration
